Open Source na nuvem

Sinta-se livre para escolher as suas tecnologias Open Source favoritas e para contratá-las na melhor plataforma da nuvem.

Sinta-se livre para escolher o mix de tecnologias que você preferir para trabalhar: encontre todas no Azure, a nuvem mais confiável do mercado.

Vantagens de negócio:

  • Preços flexíveis: Você pode ter acesso às ferramentas Open Source através de nossos preços flexíveis na nuvem. Para realizar desenvolvimentos e testes, você pode abastecer-se e jogar fora as máquinas virtuais efêmeras do Linux e pagar somente pelo que usar.

  • Plano de pré-compra Azure Compute: Caso você tenha cargas de trabalho sazonais, você pode adquirir em forma antecipada capacidade de processamento do Azure ganhando descontos.

Vantagens tecnológicas:

Microsoft Azure pode implantar um amplo leque de soluções relativas ao sistema operacional e software abertos dirigidos pela comunidade e de distribuição de Linux, como Ubuntu e SUSE; soluções impulsionadas pela comunidade, como Chef, Puppet e Docker; e outros produtos como Oracle Database e Oracle WebLogic Server.

Obtenha suporte para a sua infraestrutura como serviço (IaaS) de Linux, Java, Docker e outras plataformas de aplicativos Web como PHP, Ruby, Python, etc.

  • Desenvolva e teste componentes do Linux e open source no Azure: Você traz suas ferramentas preferidas e as competências que já possui e executa praticamente qualquer aplicativo, utilizado a origem de dados, com o sistema operacional, no seu dispositivo.

  • Agregue valor às tecnologias que interatuam bem: Aumente o valor do seu aplicativo de código aberto com a identidade e gestão de acesso mediante o Azure Active Directory ou pontos de vista nuvem, alimentado mediante o Azure Data Lake Analytics.

  • Assegure aquilo que constrói para o futuro: Na Microsoft, o código aberto é uma parte essencial do nosso enfoque inovador na nuvem.

  • Continue usando as suas licenças: No Azure, você pode manter as licenças que possui atualmente e continuar a usá-las. Não é necessário pagar novamente por aquilo que você já adquiriu.

A liberdade de escolher no Azure:
  • 30% das máquinas virtuais do Azure utilizam tecnologia Open Source
  • 70% do catálogo do Azure marketplace é Open Source.
